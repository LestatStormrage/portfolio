    About me

    Hi! My name is Ivan. I’m originally from Ukraine, live in Germany. I’ve explored different areas in IT and found web development to be the one I enjoy the most.

    Over time, I noticed that many websites are built using page builders and pre-made themes, which often results in slow performance, unnecessary complexity, and limited flexibility. I take a different approach - I build websites from scratch, writing all code manually. This allows me to create fast and secure websites with very custom design.

    Also, I know small businesses can't always afford a good website, so I offer a subscribtion model.

    FAQ

    How long does the website take from start to finish?
    This depends of the complexity of the website and the amount of content. In average, it takes between 2-6 weeks.
    Why do you write all code by hand instead of using web page builders like the others do?
    First of all, this allows me to create websites with unique designs that are not limited by the capabilities of page builders. Second - by writing all the code by hand I'm making sure that the code is clean and optimized for performance, which is often not the case with page builders. This increases load times and performance. Also, because I'm not relying on builders and there's no need to constantly updage any addons or Wordpress versions, the websites I make are literally impossible to hack - there's just nothing that can be hacked.
